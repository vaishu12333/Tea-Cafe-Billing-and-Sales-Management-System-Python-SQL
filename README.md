# Tea Cafe Billing and Sales Management System

## Project Overview

This project was developed by **Vaishnavi Patil** using Python and MySQL for managing cafe billing and customer sales data.

The system helps to store customer orders, generate bills, manage transactions, and track shop-wise sales records efficiently.

---

## Features

* Add customer billing details
* Automatic total bill calculation
* Store records in MySQL database
* Save transaction data in TXT file
* Search customer by name
* View all customer records
* Date-wise sales report
* Shop-wise summary report
* Multiple payment mode support

---
